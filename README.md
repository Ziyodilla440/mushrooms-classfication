# 🍄 Mushroom Classification

## Overview

This project uses machine learning to classify mushrooms as **edible** or **poisonous** based on their characteristics.

The project includes data preprocessing, exploratory data analysis (EDA), model training, and performance evaluation.

## Dataset

The Mushroom dataset contains various features such as:

* cap shape
* odor
* gill color
* habitat

Target classes:

* `e` = edible
* `p` = poisonous

## Steps Performed

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Label Encoding
* Data visualization
* Train-test split
* Model training and evaluation

## Models Used

* K-Nearest Neighbors (KNN)
* Decision Tree
* Random Forest
* Logistic Regression

## Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1-score

## Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Jupyter Notebook

## Conclusion

This project demonstrates the basic machine learning workflow, from data analysis to model evaluation, and helped improve practical skills in data science and classification tasks.
